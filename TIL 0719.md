### 값 비교

- is 와 == 는 다른 역할을 한다.

- is는 값과 값이 저장된 메모리 공간을 비교한다

- ==는 값만 비교한다.

- 예를 들어

[링    크](https://itholic.github.io/python-identity-equality/)

### Float를 문자로

```python
str(3.5)

```

하면 '3.5'이나온다



## Float를 integer로

```python
int(3.5)
```

하면 3이 나온다. 반올림이 아닌 버림 기준인듯?



## 반올림

```py
round()
```

round함수로 반올림 할 수 있다. 단 딱 x.5일 경우 반올림이 정확하지 않다. 짝수에 가깝게 변환한다고 하는데 옵션 설정 인자로 조정가능한 듯 하다.
